# AgenticNeuroSymboicFramework
An Agentic Neuro-Symbolic Framework for Explainable Chronic Disease Monitoring in Smart Healthcare

## Overview
A hybrid AI system combining **LSTM/Transformer + symbolic rules** to predict diabetes risk and generate actionable alerts.

## Features
- Features (glucose, insulin, nutrition, activity, etc.)
- Explainable rule-based reasoning
- Neuro-symbolic fusion for safer predictions
- Agent decisions: SAFE / WATCH / ALERT / EMERGENCY

## Outputs
- Risk predictions & explanations
- Confusion matrix, classification report
- Alerts and Visualizations

## Results
- Accuracy: ~93–97%
- Improved reliability with safety overrides
